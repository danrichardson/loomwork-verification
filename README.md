# Coastal Kitchen

A food and recipe site focused on seafood and coastal cooking. Built with [Astro](https://astro.build), deployed to Cloudflare.

## About

Coastal Kitchen covers seafood recipes, cooking techniques, and pantry essentials for coastal cuisine — from clam chowder and grilled salmon to shucking oysters and making fish stock.

## Content

- **[About](/about)** — About Coastal Kitchen
- **[Recipes](/recipes)** — Featured seafood recipes
- **[Techniques](/techniques)** — Coastal cooking techniques
- **[Pantry](/pantry)** — Pantry essentials for coastal cooking

## Development

```bash
npm install
npm run dev       # http://localhost:4321
npm run build     # Build for production
```

## Deployment

Deployed to Cloudflare Workers/Pages. Push to `main` to trigger a build.

## Built With

This site is built on the [Loomwork](https://github.com/danrichardson/loomwork) Astro starter framework.

```bash
# Pull framework updates
git fetch loomwork
git merge loomwork/main
```
